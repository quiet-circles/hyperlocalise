`hyperlocalise` helps you generate local translation drafts, sync with your TMS, and track what still needs review.

## Who this is for

Use this CLI if you:

- keep translation files in your repository,
- want AI-generated drafts as a starting point,
- still need human curation in POEditor, Lokalise, or Crowdin.

## Core workflow

```text
init -> configure i18n.jsonc -> run --dry-run -> run -> sync push -> curate in TMS -> sync pull -> status
```
