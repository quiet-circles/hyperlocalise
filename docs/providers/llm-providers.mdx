---
title: "LLM providers"
description: "Configure OpenAI, Azure OpenAI, Anthropic, Gemini, Bedrock, LM Studio, Ollama, or Groq for translation generation profiles."
---

Supported `llm.profiles.*.provider` values:

- `openai`
- `azure_openai`
- `anthropic`
- `gemini`
- `lmstudio`
- `groq`
- `ollama`
- `bedrock`

Required fields per profile:

- `provider`
- `model`
- `prompt`

Prompt variables:

- `{{source}}`
- `{{target}}`
- `{{input}}`

## OpenAI example

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "openai",
        "model": "gpt-5.2",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

```bash
export OPENAI_API_KEY="your-openai-api-key"
```

## Azure OpenAI example

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "azure_openai",
        "model": "gpt-4.1-mini",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

```bash
# Example: https://<resource>.openai.azure.com/openai/v1
export AZURE_OPENAI_BASE_URL="https://<resource>.openai.azure.com/openai/v1"
export AZURE_OPENAI_API_KEY="your-azure-openai-api-key"
```

export AWS_ACCESS_KEY_ID="your-access-key-id"
