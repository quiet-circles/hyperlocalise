---
title: "LLM providers"
description: "Configure OpenAI or LM Studio for translation generation profiles."
---

Supported `llm.profiles.*.provider` values:

- `openai`
- `lmstudio`

Required fields per profile:

- `provider`
- `model`
- `prompt`

Prompt variables:

- `{{source}}`
- `{{target}}`
- `{{input}}`

## OpenAI example

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "openai",
        "model": "gpt-5.2",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

```bash
export OPENAI_API_KEY="your-openai-api-key"
```

## LM Studio example

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "lmstudio",
        "model": "qwen2.5-7b-instruct",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

```bash
export LM_STUDIO_BASE_URL="http://127.0.0.1:1234/v1"
export LM_STUDIO_API_KEY="lm-studio"
```
