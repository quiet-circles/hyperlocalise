---
title: "LLM providers"
description: "Configure OpenAI, LM Studio, or Groq for translation generation profiles."
---

Supported `llm.profiles.*.provider` values:

- `openai`
- `lmstudio`
- `groq`

Required fields per profile:

- `provider`
- `model`
- `prompt`

Prompt variables:

- `{{source}}`
- `{{target}}`
- `{{input}}`

## OpenAI example

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "openai",
        "model": "gpt-5.2",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

```bash
export OPENAI_API_KEY="your-openai-api-key"
```

## LM Studio example

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "lmstudio",
        "model": "qwen2.5-7b-instruct",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

```bash
export LM_STUDIO_BASE_URL="http://127.0.0.1:1234/v1"
export LM_STUDIO_API_KEY="lm-studio"
```


## Groq example

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "groq",
        "model": "llama-3.3-70b-versatile",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"

    }
  }
}
```

```bash
export GROQ_BASE_URL="https://api.groq.com/openai/v1"
export GROQ_API_KEY="your-groq-api-key"
```
