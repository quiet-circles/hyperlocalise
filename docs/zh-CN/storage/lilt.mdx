---
title: "Lilt adapter"
description: "Configure Lilt for remote translation pull/push synchronization."
---

##必需配置
```jsonc
{
  "storage": {
    "adapter": "lilt",
    "config": {
      "projectID": "123",
      "apiTokenEnv": "LILT_API_TOKEN"
    }
  }
}
```

##令牌解析
1. 由…命名的环境变量`storage.config.apiTokenEnv`
2. 回退：`LILT_API_TOKEN`

`apiToken`配置中不支持内联。请仅使用环境变量。
##可选配置字段
- `targetLanguages`
- `timeoutSeconds` (默认为`30`)
- `pollIntervalMs` (默认为`1000`)
- `maxPolls` (默认设置为`60`)

##拉动行为
-为每个区域启动一个导出作业-轮询导出状态直到作业完成-下载 JSON 工件(单个文件或压缩包)
-规范条目并跳过空的键/语言/值行
##常见问题
-缺失标记：export`LILT_API_TOKEN`在同一个 shell 会话中-缺少项目ID：设置`storage.config.projectID`
-超时或导出缓慢：增加`timeoutSeconds`, `pollIntervalMs`，或`maxPolls`
