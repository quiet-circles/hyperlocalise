## 必需配置

```jsonc
{
  "storage": {
    "adapter": "lilt",
    "config": {
      "projectID": "123",
      "apiTokenEnv": "LILT_API_TOKEN"
    }
  }
}
```

## 令牌解析

1. 由以下名称命名的环境变量 `storage.config.apiTokenEnv`
2. 回退： `LILT_API_TOKEN`

`apiToken` 不支持在配置中内联。仅使用环境变量。

## 可选配置字段

- `targetLanguages`
- `timeoutSeconds` (默认为 `30`)
- `pollIntervalMs` (默认为 `1000`)
- `maxPolls` (默认为 `60`)

## 拉取行为

- 为每个区域设置启动一个导出作业
- 轮询导出状态，直到作业完成
- 下载 JSON 工件 (单个文件或 ZIP)
- 规范化条目并跳过 key/locale/value 为空的行

## 常见问题

- 缺少令牌：export `LILT_API_TOKEN` 在同一个 shell 会话中
- 缺少项目 ID：设置 `storage.config.projectID`
- 超时或导出缓慢：增加 `timeoutSeconds`, `pollIntervalMs`，或 `maxPolls`
