---
title: "版本、更新和完成"
description: "检查已安装的版本，自行更新 CLI，并生成 shell 自动补全脚本。"
---

## 版本

```bash
hyperlocalise version
```

预期输出格式：

```text
hyperlocalise: <version>
```

## 更新

更新到最新发布版本：

```bash
hyperlocalise update
```

固定特定的发布标签：

```bash
hyperlocalise update v1.2.3
```

## 完成

使用内置-在 Cobra 补全命令中：

```bash
hyperlocalise completion --help
```

然后为你的 shell 生成并安装补全脚本。
