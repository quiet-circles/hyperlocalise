title: "Performance"
description: "Improve runtime and stability on large localization projects."
---

##减少可避免的工作
-依赖锁-文件跳过于`run`
-将非常大的映射拆分成更小的桶-通过重复限制区域范围`--locale`

##提高同步可靠性
-设置适配器`timeoutSeconds`在处理大型项目时-每次运行-本地同步作业而不是全风扇-在需要时退出-使用 JSON 输出以简化自动重试逻辑
##诊断热点
-比较`planned_total`对抗`executable_total`
-检查重复失败情况`target`和`key`
-检查您的提供商的网络/API 速率限制
