## 最小配置模式

使用一个存储桶和一个组开始：

- `locales.source`：您的源语言环境 (例如 `en-US`)
- `locales.targets`: 一个目标区域 (例如 `es-ES`)
- `buckets.ui.files`: 将源路径模板映射到目标路径模板
- `groups.default`：附加 `targets` 和 `buckets`

有关完整字段，请参见 [i18n 配置](/configuration/i18n-config).

在此设置中，翻译来源于 `packages/ui/lang` 并生成到 `apps/web/lang`.
