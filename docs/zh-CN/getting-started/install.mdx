## 先决条件

- macOS 或 Linux
- 以下之一：安装脚本、预构建二进制文件、 `.deb`/`.rpm` 包，或 Go 工具链

## 使用引导脚本安装

使用稳定的 bootstrap URL：

```bash
curl -fsSL https://raw.githubusercontent.com/quiet-circles/hyperlocalise/main/install.sh | bash
```

固定到特定版本：

```bash
curl -fsSL https://raw.githubusercontent.com/quiet-circles/hyperlocalise/main/install.sh | VERSION=v1.2.3 bash
```

安装脚本也会附加到每个 GitHub 发布版本中，因此你可以直接从发布资源中查看或下载带标签的版本。

## 从源代码安装 (Go)

```bash
go install github.com/quiet-circles/hyperlocalise@latest
```

## 从代码仓库构建

```bash
make bootstrap
make install
```

## 验证安装

```bash
hyperlocalise version
hyperlocalise --help
```

预期输出包含可用的命令： `init`, `run`, `status`, `sync`, `version`，并且 `update`.

## 安装 Hyperlocalise 技能 (skills.sh)

安装：

```bash
npx skills add https://github.com/quiet-circles/hyperlocalise --skill hyperlocalise
```

这使用了 [skills.sh](https://skills.sh) 通过安装程序 `npx`.

## 升级

使用内置的-在 self 中-更新命令：

```bash
hyperlocalise update
```

固定特定版本：

```bash
hyperlocalise update v1.2.3
```

你仍然可以重新-如需，可运行原始安装方法。
