## Chỉnh sửa trên GitHub

1. mở trang,
2. nhấp vào chỉnh sửa,
3. gửi yêu cầu kéo.

## Quy trình làm việc tài liệu nội bộ

1. sao chép kho lưu trữ,
2. cài đặt Mintlify CLI (`npm i -g mint`),
3. chạy `mint dev` trong `docs/`,
4. xác thực liên kết với `mint broken-links`,
5. mở yêu cầu kéo.

## Tiêu chuẩn viết

- sử dụng thể chủ động và ngôi thứ hai,
- giữ cho các ví dụ có thể chạy được,
- mở đầu mỗi trang bằng mục tiêu và các yêu cầu tiên quyết,
- giữ các cờ lệnh căn chỉnh với đầu ra trợ giúp CLI hiện tại.
