##Chỉnh sửa trên GitHub
1. mở trang,2. bấm chỉnh sửa,3. gửi yêu cầu kéo.
##Quy trình tài liệu nội bộ
1. sao chép kho lưu trữ,2. cài đặt Mintlify CLI(`npm i -g mint`),
3. chạy`mint dev`trong`docs/`,
4. xác thực liên kết với`mint broken-links`,
5. mở pull request.
##Tiêu chuẩn viết bài
-sử dụng thể chủ động và ngôi thứ hai,-giữ cho ví dụ có thể thực thi,-bắt đầu mỗi trang với mục tiêu và các điều kiện tiên quyết,-giữ các cờ lệnh căn chỉnh với đầu ra trợ giúp CLI hiện tại.
