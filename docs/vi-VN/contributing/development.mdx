##Yêu cầu tiên quyết
-Go 1.26+
- `git`
- `make`có sẵn trong shell của bạn
##Sao chép kho lưu trữ
```bash
git clone https://github.com/quiet-circles/hyperlocalise.git
cd hyperlocalise
```

##Công cụ và mô-đun Bootstrap
```bash
make bootstrap
```

Điều này tải xuống các module Go và cài đặt chuỗi công cụ lint đã được ghim.
##Vòng lặp phát triển hàng ngày
```bash
make fmt
make lint
make test
make check-build
```

##Chạy CLI tại chỗ
```bash
make run
# or
go run -ldflags "-X main.version=dev" main.go
```

##Cài đặt nhị phân cục bộ
```bash
make install
hyperlocalise version
```

##Vị trí mã phím
- `cmd/`: Lệnh Cobra, cờ và hành vi đầu ra CLI- `internal/config/`: `i18n.jsonc`phân tích cú pháp và xác thực- `internal/i18n/runsvc/`: `run`luồng kế hoạch và thực thi- `internal/i18n/syncsvc/`: đồng bộ điều phối và logic xung đột- `internal/i18n/storage/`: hợp đồng bộ chuyển đổi và tích hợp nhà cung cấp
##Trước khi mở một PR
```bash
make precommit
```

Việc này chạy định dạng, kiểm tra lỗi cú pháp, kiểm tra và xây dựng trong cùng một quy trình được sử dụng bởi các cộng tác viên.
