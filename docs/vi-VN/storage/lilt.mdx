## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "lilt",
    "config": {
      "projectID": "123",
      "apiTokenEnv": "LILT_API_TOKEN"
    }
  }
}
```

## Phân giải token

1. biến môi trường được đặt tên bởi `storage.config.apiTokenEnv`
2. dự phòng: `LILT_API_TOKEN`

`apiToken` Không hỗ trợ cấu hình nội tuyến. Chỉ sử dụng biến môi trường.

## Các trường cấu hình tùy chọn

- `targetLanguages`
- `timeoutSeconds` (mặc định là `30`)
- `pollIntervalMs` (mặc định là `1000`)
- `maxPolls` (mặc định là `60`)

## Hành vi kéo

- bắt đầu một tác vụ xuất cho mỗi ngôn ngữ môi trường‬
- Thăm dò trạng thái xuất cho đến khi tác vụ hoàn tất
- tải xuống các hiện vật JSON (tệp đơn lẻ hoặc ZIP)
- chuẩn hóa các mục nhập và bỏ qua các hàng khóa/ngôn ngữ/giá trị trống

## Các sự cố thường gặp

- thiếu token: xuất `LILT_API_TOKEN` trong cùng một phiên shell
- thiếu ID dự án: đặt `storage.config.projectID`
- hết thời gian chờ hoặc xuất chậm: tăng `timeoutSeconds`, `pollIntervalMs`, hoặc `maxPolls`
