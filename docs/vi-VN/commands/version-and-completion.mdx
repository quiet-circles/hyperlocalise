---
title: "version and completion"
description: "Check installed version and generate shell completion scripts."
---

##Phiên bản
```bash
hyperlocalise version
```

Định dạng đầu ra mong đợi:
```text
hyperlocalise: <version>
```

##Hoàn thành
Sử dụng tích hợp sẵn-trong lệnh hoàn thành Cobra:
```bash
hyperlocalise completion --help
```

Sau đó tạo và cài đặt các tập lệnh hoàn thành cho shell của bạn.
