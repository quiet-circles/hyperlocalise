## Dạng CLI

```bash
hyperlocalise [command]
```

Các lệnh chính:

- `init`
- `run`
- `eval`
- `status`
- `sync pull`
- `sync push`
- `version`
- `completion`

## Bắt đầu điểm theo nhiệm vụ

- Cài đặt đầu tiên: [khởi tạo](/commands/init)
- Tạo bản nháp cục bộ: [chạy](/commands/run)
- Thí nghiệm chất lượng và kiểm soát truy cập: [đánh giá](/commands/eval)
- Báo cáo tiến độ: [trạng thái](/commands/status)
- Kéo bản dịch được tuyển chọn: [đồng bộ kéo](/commands/sync-pull)
- Đẩy các thay đổi cục bộ: [đồng bộ đẩy](/commands/sync-push)
